26 oct 2020

Dashboard Screen Added.

Dashboard with quick start links and quick stats added.

Admin Mobile Support Added

Admin is now mobile friendly.

10 oct 2020

Selection area moved.

The Frame/Background/Filter/Sticker selection area has been moved below the booth artwork on the user interface. This is for Square layouts only.

6 oct 2020

Gallery Approval.

There is now an option to ask users for permission to add their photos to the Gallery. You can enable/disable this as well as change the text for the question in the Sharing tab.

Dropbox Integration.

Dropbox has been added via the Integration menu. You can now sync your Dropbox account to Virtual Booth in order to download and print incoming photos. Enable/Disable Dropbox sync on a per-event basis in the Sharing tab.
